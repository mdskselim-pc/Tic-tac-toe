## How to Play 

Tic Tac Toe is a 2-player game played on a 3 × 3 grid. Players take turns placing their mark (X or O) in an empty square. The goal is to get exactly 3 of your marks in a row (horizontally, vertically, or diagonally). If all 9 squares are filled and no one has 3 in a row, the game is a tie. You can reset game after game finishes.
